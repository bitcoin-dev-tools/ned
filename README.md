# ned

An [`nvf`](https://github.com/NotAShelf/nvf#get-started) neovim flake for Bitcoin Core development.

## Features

LSPs configured for use with:

- C++
- Rust
- Python
- Nix

Includes:

- Nice vim defaults
- Snippets (blink-cmp)
- Git helpers
- Theme
- Statusline
- Treesitter
- Formatters

## Platforms

Works on Linux or MacOS, on both x64 or aarch64.
